# AstraTrip Architecture

## Tech Stack
- **Frontend:** React, Vite, TypeScript, Tailwind CSS, shadcn/ui
- **Backend:** FastAPI, Python, Pydantic
- **Database:** Supabase (PostgreSQL)
- **AI:** Gemini 2.5 Pro API (via LangGraph/Agentic pattern)

- `database/`
